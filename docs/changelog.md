- `momentum_low` 命中率 25.6%，相对基准发生率 22.9%，`lift = 1.119`，具备弱预警增益，可作为候选预警因子。

- 动量因子只能作为弱预警补充，不能单独驱动高等级预警；后续仍需补充估值、市场宽度、资金流向等更独立的信息源。

## [1.0.3] 多因子融合探索（已回滚至 1.0.2 / 失败记录）2026-06-30

### 尝试内容
- 新建 `src/warning.py`：多因子融合规则引擎，以自相关因子为预警主力，能量比/高频占比为温度计叠加升级，输出四级预警（无/关注/警惕/危险）。
- `main.py` 新增预警模式，`plot.py` 新增 `plot_warning_signals` 全景图。

### 失败根因：六个经验教训

1. **混淆了"状态"和"事件"**
   早期版本把 `autocorr < 0` 当作逐日持续状态报警（一旦转负就连报数百天），导致预警天数过多、重复告警严重。后续引入跳变事件 + 冷却期后，虽然告警次数下降，但并没有从根本上提升每次告警的质量。

2. **三个因子本质同源，信息量没有增加**
   能量比（收益率平方的短/长期比）、高频占比（收益率 FFT 高频段能量占比）、自相关（收益率序列自相关系数）——三者全部基于同一个 `close.pct_change()` 序列派生，只是同一把尺子的不同刻度。它们的叠加没有引入任何独立维度的信息。

3. **温度计因子无领先性，叠加不产生新价值**
   evaluate.py 已验证能量比/高频占比命中率 0.85x/0.87x，属于波动的同期产物而非领先信号。把它们叠加到自相关上，只是将预警等级做了颜色叠加，并没有改变"能不能预警"这个核心问题。

4. **缺失真正意义上的多因子**
   roadmap 中规划的传统金融多因子（估值 PE/PB、动量、质量 ROE、贝塔、资金流向等）一个都没有引入。融合层实质上是自相关单因子带两个同源衍生指标在自娱自乐，算不上"多因子融合"。

5. **规则阈值是拍脑袋的，没有经过回测验证**
   `energy > 1.5`、`hfr > 0.6`、`ac_threshold = 0.05` 都是硬编码，没有用 evaluate.py 的验证框架衡量这些阈值组合对命中率的实际贡献。

6. **正确的融合方向应该先补因子，再想融合**
   多因子融合的前提是有"多"个独立信息源的因子。应该先回到 roadmap 第四阶段，引入估值/动量/质量/贝塔等真正的传统金融因子作为独立维度，各自有独立的预警逻辑和验证分数，再考虑规则融合或打分聚合。

### 结论
- 代码已回滚到 `1.0.2` 阶段，`src/warning.py` 已删除。
- 融合层的正确起点不是调参调阈值，而是先引入真正的多因子（跨维度、独立信息源），再谈融合。

## [1.0.2] 2026-06-29

### 新增
- 信号处理因子包（`signals/`），将电子信息领域的信号分析方法引入量化预警：
  - `spectrum.py`：频域高频能量占比因子（FFT，`numpy.fft` 实现，不依赖 scipy）
  - `energy.py`：时域短期/长期能量比因子
  - `autocorr.py`：滚动自相关因子（趋势持续性 / 动能衰竭）
  - `evaluate.py`：预警有效性评估框架 + 布林带/RSI 基准对比
- `plot.py` 新增 `plot_signal_factors`：收盘价与三因子对照图

### 验证结论（沪深300全历史，重要）
- **系统定位明确为中长期风险预警**（未来60交易日视角），不做短期涨跌预测。
- **命中口径**：预警后未来60交易日内最大回撤 > 15% 即命中。
- **关键发现**：
  - 自相关转负因子命中率 27.1%，跑赢基准（提升 1.45x），优于 RSI(1.12x)、布林带(0.96x)。
  - 能量比 / 高频占比等**幅度类因子是波动的同期产物，无预警领先性**（0.85x / 0.87x）。
  - 结论：**趋势结构类因子（自相关）显著优于能量幅度类因子**。后者转为"当前波动状态展示"用途。
- **真实案例**：自相关转负在 2015 股灾（4500点顶部提前预警，后续回撤 -31%）、2021 抱团顶均成功预警；2018 慢熊阴跌型漏报 → 印证单因子有盲区，需多因子融合互补。
- **方法论教训**：定性看图易把"同步"误判为"领先"，必须有定量验证框架；命中口径需区分"波动水平"与"波动跳升/回撤"。

### 变更
- 三个因子默认窗口统一调整为中长期配置（约 250 交易日 ≈ 1 年），与验证有效的口径一致。

### 设计定位确认
- 系统核心定位调整为**多维度市场结构展示 + 中长期风险预警**，不以预测短期涨跌为目标。
- 因子分工形成两层：
  - **温度计层**：高频占比 + 能量比，回答"市场现在处于什么状态"（高噪声/高波动），供可视化与 LLM 解读使用
  - **预警层**：自相关趋势动能 + 后续传统多因子，回答"是否有中长期结构性风险"，驱动四级预警等级
- 根本逻辑：能量类因子描述"现在有多乱"（同期），自相关类因子捕捉"趋势结构正在瓦解"（领先）。

### 下一步
- 第四阶段：多因子融合与预警层（规则引擎，输出 无/关注/警惕/危险 四级），以自相关为预警主力、幅度类因子作状态展示。
